Each stage of the 4th-order Butterworth uses a unity-gain Sallen-Key low-pass topology. The key is that C_feedback connects from the op-amp output back to the R-R junction node, not to ground.

✓ What's Correct The R-R series chain topology is right for both stages. R3→R4 in series with C13 teeing off the junction, and R5→R6 in series with C15 teeing off the junction — that IS the correct Sallen-Key passive arrangement. The component values are also correct (1.62k/5.76k/11n/1.1n for Stage 1, 17.4k/536Ω/11n/1.1n for Stage 2). The PGA113 wiring is correct — all hierarchical labels connect properly.
⚠ Schematic Audit Update (March 2026): C14 Pin2 is FLOATING, not grounded A programmatic audit of the KiCad schematic found that C14.Pin2 is unconnected (floating), not wired to GND as originally reported. In fact, no GND power symbols exist on Sheet C at all. All four filter capacitor second pins (C13, C14, C15, C16) are floating. The earlier claim that "C14 is wired to GND" was incorrect.
⚠ LTC6242 signal pins are entirely unconnected All signal pins (+IN_A, -IN_A, OUT_A, +IN_B, -IN_B, OUT_B) have net labels placed nearby in the schematic, but the labels are isolated — they do not connect to any wire segment. The op-amp is placed but electrically invisible. Only power pins (V+, V-) and NC pins appear to have connections.
⚠ Stage 1 → Stage 2 connection missing R5.Pin1 is floating. The Stage1_OUT label exists but one instance is isolated from the wire network.
⚠ No GND power symbols on Sheet C GND symbols must be added before shunt caps (C13, C15) can be connected to ground. This is a prerequisite for all filter wiring fixes.
